# 题目复盘

这个目录用于保存有复习价值的题目，不再只等同于“错题本”。

适合记录的题目包括：

1. 错题：暴露概念、计算、审题或方法选择问题。
2. 典型题：代表某一类题型的标准入口。
3. 易混题：容易和相近概念、相近方法混淆。
4. 好方法题：题目本身不一定难，但方法值得迁移。
5. 综合题：能串联多个章节或多个知识点。

综合题先放到主考点所在学科；如果以后综合题明显变多，再单独建立综合题入口。

## 使用原则

1. 不追求题量，只记录“以后值得再看”的题。
2. 每道题重点写清楚“为什么值得记录”和“下次如何识别”。
3. 如果是错题，要写错因；如果不是错题，要写它的典型性或方法价值。
4. 题目复盘要尽量链接回对应章节、概念卡片或方法笔记。
