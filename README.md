# 考研数学 Obsidian 知识库

本仓库主要作为 Obsidian Vault 使用，用来沉淀中文考研数学笔记、方法归纳、题库材料和错题记录。

GitHub 只作为备份和版本管理环境，不再作为主要阅读入口。日常复习建议直接在 Obsidian 中打开本仓库目录，通过双向链接组织知识点和题型。

Codex 用于批量整理、格式修复、生成题目、维护题库结构和检查仓库质量。

## 主复习入口

- `高等数学/`：高等数学专题笔记。
- `线性代数/`：线性代数专题笔记。
- `概率论/`：概率论与数理统计专题笔记。
- `方法库/`：可跨专题复用的解题方法。
- `错题/`：错题模板、错题复盘和阶段性回顾。
- `题库/`：面向 Obsidian 阅读的题目索引和专题题单。

当前可从 `高等数学/积分.md` 开始复习，并结合：

- `方法库/换元积分法.md`
- `方法库/分部积分法.md`
- `方法库/对称性积分.md`
- `方法库/反常积分判别.md`

## 辅助工具

以下目录保留为辅助工具，不作为主复习入口：

- `problems/`：结构化 JSON 题库，便于脚本读取和测试。
- `scripts/`：自动生成练习卷、批量维护内容的脚本。
- `tests/`：题库格式和生成逻辑的测试。
- `topics/`：早期专题笔记归档，后续优先迁移到中文 Obsidian 目录。

## 使用方式

在 Obsidian 中打开本仓库目录，优先阅读中文目录下的笔记，并使用 `[[双向链接]]` 串联专题、方法和错题。

修改题库或维护脚本后，运行：

```bash
pytest
```

如果修改了生成练习卷的逻辑，再运行：

```bash
python scripts/generate_quiz.py
```

默认会读取 `problems/integrals.json`，并生成 `quiz.md`。
