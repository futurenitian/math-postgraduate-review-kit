# 考研数学 Obsidian 知识库

这个仓库现在只做两件事：整理考研数学复习笔记，记录和复盘错题。

建议在 Obsidian 中打开本目录，从各学科总览进入专题笔记，再用 Obsidian 双向链接串联概念、方法和错题。

## 当前结构

- `高等数学/`：高等数学专题笔记，目前重点是 积分。
- `线性代数/`：线性代数专题笔记，目前包含 线性代数总览、行列式、矩阵。
- `概率论/`：概率论与数理统计笔记，目前是总览占位，后续逐步补充。
- `方法库/`：跨专题复用的解题方法，例如 换元积分法、分部积分法、初等变换。
- `错题/`：错题记录模板和后续错题复盘。

## 使用建议

1. 新知识点放入对应学科目录。
2. 通用解题套路放入 `方法库/`。
3. 做错的题统一按 错题模板 记录到 `错题/`。
4. 重要概念、方法和错题之间尽量使用 Obsidian 双向链接。
5. 保持笔记适合直接复习，不再维护任何非笔记类功能。
