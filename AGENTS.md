# 协作说明

本仓库用于维护中文考研数学复习资料。后续协作者或智能代理在修改仓库时，请遵守以下约定。

## 内容约定

- 所有面向学习者的说明、题目、解析和文档使用中文。
- 题库文件保持结构化，优先使用 JSON，便于脚本读取和测试。
- 新增专题时，建议同时添加：
  - `topics/专题名.md`
  - `problems/专题名.json`
  - 对应的格式测试或扩展现有测试

## 题目字段

每道题至少包含以下字段：

- `id`：唯一题号
- `topic`：专题名称
- `difficulty`：难度，建议使用 `基础`、`中等`、`提高`
- `question`：题干
- `answer`：答案
- `solution`：解析
- `tags`：标签列表

## 测试要求

修改题库后，请运行：

```bash
pytest
```

如果修改了生成练习卷的逻辑，请同时运行：

```bash
python scripts/generate_quiz.py
```

## Git 约定

- 不修改 Git 配置。
- 不提交临时文件、缓存文件或与复习系统无关的产物。
- 保持提交内容聚焦，避免无关重构。
