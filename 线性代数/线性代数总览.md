# 线性代数总览

线性代数是考研数学中结构性很强的一部分。刚开始复习时，不必急着把所有结论背完，先把对象、运算和基本方法弄清楚。

主要模块包括：

1. [[行列式]]
2. [[矩阵]]
3. 向量组
4. 线性方程组
5. 特征值与特征向量
6. 二次型

这一步先展开前两个模块：[[行列式]] 和 [[矩阵]]。

## 行列式解决什么问题

[[行列式]] 通常用来判断方阵是否可逆、计算特殊方阵的数值特征，并为后面的特征值、方程组和二次型打基础。

考研中，行列式常见考法是直接计算、利用性质化简，以及通过 $|A|\ne 0$ 判断矩阵是否可逆。

## 矩阵解决什么问题

[[矩阵]] 是线性代数的主要语言。线性方程组、向量变换、秩、逆矩阵、特征值等内容，几乎都要通过矩阵表达。

学习矩阵时，先掌握加法、数乘、乘法、转置和逆矩阵，再逐步过渡到 [[初等变换]] 和 [[矩阵秩的计算]]。

## 为什么矩阵秩很重要

矩阵秩 $r(A)$ 可以理解为矩阵中真正独立的信息量。它会连接很多判断：

1. 方程组是否有解。
2. 解是否唯一。
3. 向量组是否线性相关。
4. 矩阵能否化简到标准形。

## 后续学习连接

学完 [[矩阵]] 和 [[矩阵秩的计算]] 后，后续会自然进入向量组和线性方程组。

向量组关注“哪些向量是独立的”，线性方程组关注“方程有没有解、解有多少”。这两个问题最终都会回到矩阵秩。
