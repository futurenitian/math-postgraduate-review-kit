# 矩阵秩的计算

矩阵秩 $r(A)$ 可以直观理解为矩阵中“真正独立的信息量”。入门阶段，先把它看成矩阵化成阶梯形后非零行的个数。

相关笔记：[[矩阵]]、[[初等变换]]

## 常用计算方法

**1. 初等行变换化阶梯形：**

通过 [[初等变换]] 把矩阵化成阶梯形矩阵。

**2. 找非零行个数：**

阶梯形矩阵中非零行的个数，就是矩阵的秩。

## 常见结论

1. 若 $A$ 是 $m\times n$ 矩阵，则 $r(A)\le \min(m,n)$。
2. 初等变换不改变矩阵的秩。
3. $n$ 阶可逆矩阵的秩等于 $n$。
4. 零矩阵的秩为 $0$。

## 常见题型

1. 求具体矩阵的秩。
2. 求含参数矩阵的秩。
3. 利用秩判断线性方程组解的情况。
4. 结合 [[行列式]] 判断方阵是否满秩。

## 入门提醒

求秩时，目标不是把矩阵算到最复杂，而是尽快化成能看出非零行个数的阶梯形。

含参数矩阵要特别注意：某个位置能不能作为主元，往往取决于参数是否为 $0$。
