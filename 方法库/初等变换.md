# 初等变换

初等变换是线性代数中最常用的化简方法。它可以用来计算行列式、求矩阵的秩、求逆矩阵，也会连接线性方程组。

相关笔记：[[行列式]]、[[矩阵]]、[[矩阵秩的计算]]

## 初等行变换

初等行变换有三种形式：

1. 交换两行。
2. 某一行乘以非零常数 $k$。
3. 某一行加上另一行的 $k$ 倍。

## 初等列变换

初等列变换也有三种形式：

1. 交换两列。
2. 某一列乘以非零常数 $k$。
3. 某一列加上另一列的 $k$ 倍。

## 在考研中的作用

1. 化简矩阵。
2. 求秩。
3. 求逆矩阵。
4. 解线性方程组。
5. 计算或化简行列式。

## 易错点

1. 对 [[行列式]] 做初等变换时，要注意行列式值是否变化。
2. 对 [[矩阵秩的计算]] 来说，初等行变换不改变秩。
3. 求逆矩阵时，左右两边要同步做行变换。
4. 行变换和列变换不要在同一步计算中随意混用。
